✦ 제우스 오라클 관리자 ✦

입금 확인 후 유저에게 권한 토큰 발급

🔒 관리자 로그인

📋 권한 부여

사용 절차:
1. 카카오뱅크/토스 입금 확인
2. 입금 금액에 맞는 플랜 선택
3. [권한 부여] 클릭 → 토큰 생성
4. 토큰을 유저에게 전달 (카톡/문자)

🔍 토큰 검증 (테스트용)

📋 입금 신고 명단 (사후 검증)

사용 절차:
1. [명단 새로고침] → 자동 토큰 발급 받은 유저 목록 조회
2. 카카오뱅크 앱에서 실제 입금 내역 대조
3. 입금 확인 → [✓ 확인] 버튼 클릭
4. 의심스러운 유저 → [🚫 차단] 버튼 클릭
